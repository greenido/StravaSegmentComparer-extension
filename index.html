Strava Segment Comparator Chrome Extension

This Chrome extension allows you to compare segments between two Strava activities and presents the data in a sortable, filterable table.

Download Extension Files

Features

  • Compare segment data between two Strava activities
  • View time differences and speed differences for each segment
  • Sort and filter the comparison data
  • Save comparison results for later viewing
  • Export results as CSV

Installation Instructions

Step 1: Download the Extension Files

Click the "Download Extension Files" button above. This will take you to a directory where you can download all the extension files.

Download each file to the same folder on your computer. Maintain the same folder structure with the 'lib' and 'icons' subfolders.

Step 2: Load the Extension in Chrome

  1. Open Google Chrome
  2. Navigate to chrome://extensions/
  3. Enable "Developer mode" using the toggle in the top right corner
  4. Click "Load unpacked"
  5. Select the folder where you saved the extension files

Note: The extension should now appear in your Chrome toolbar. You might need to click the puzzle piece icon to see and pin it.

How to Use

Step 1: Open Strava Activities

Navigate to a Strava activity page that contains segments. The extension will automatically detect the current activity URL.

Step 2: Compare Activities

Click the extension icon in your browser toolbar to open the popup.

  1. Enter two Strava activity URLs in the input fields
  2. Click "Compare Activities"
  3. The extension will open tabs to extract segment data from both activities
  4. Once complete, the results will be displayed in a table

Step 3: Analyze Results

The table shows segment name, times, speeds, and the differences between the two activities:

  • Green values indicate faster times or higher speeds
  • Red values indicate slower times or lower speeds
  • Use the search field to filter for specific segments
  • Click column headers to sort by that column

Step 4: Save or Export Results

You can:

  • Click "Save Results" to store the comparison in your browser's localStorage
  • Click "Load Saved" to retrieve previously saved comparison data
  • Click "Export as CSV" to download the data as a CSV file for further analysis

Privacy Note: This extension operates entirely in your browser. No data is sent to any server.

Files in this Extension

  • manifest.json - Extension configuration
  • popup.html - The extension popup interface
  • popup.js - Main functionality for the popup
  • popup.css - Styling for the popup
  • content-script.js - Script that extracts segment data from Strava
  • background.js - Background service worker
  • utils.js - Utility functions
  • lib/ - Libraries used by the extension
